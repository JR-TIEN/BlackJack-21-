# BlackJack-21-
This is a poker game (Blackjack) for a single player.
Gaming Process:
First, input the amount you want to bet, then the game will start.During the game, the player has 4 possible actions depending on the card status:
  1.Hit: Take another card.
  2.Split: If player get two cards with the same value, player can split them into separate hands and add the same bet.
  3.Double: Double the bet and take the last card (becoming a stand).
  4.Stand: Do not take any more cards.
The goal is to make the total value of the cards in your hand as close to 21 as possible, without exceeding it, in order to win the game.
